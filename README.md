## Descrição do Projeto

Sistema desenvolvido em React.JS com três itens de menu:

- Álbuns
- Postagens
- Tarefas (Todo List)

### Funcionlidades

- Álbuns
  - Cadastrar
  - Editar
  - Excluir
- Postagens
  - Cadastrar
  - Editar
  - Excluir
- Tarefas (Todo List)
  - Cadastrar
  - Editar
  - Excluir
  - Filtragem

## Bibliotecas Usadas

### UI/UX

- Ant Design (UI/UX Framework)
- Node SASS

### Storage

- Redux
- Redux Persist

### Teste

- Jest
- Testing Library

### Gráficos

- React Chart 2

## Pré requisitos

- Node versão superior à 14

## Instalação

- $ yarn

## Testes

- $ yarn test

### Para verificar a cobertura de testes:

- $ yarn test --coverage
